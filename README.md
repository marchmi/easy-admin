## 安装步骤

```
git clone https://github.com/marchmi/easy-admin.git
cd easy-admin
npm install         // npm install --legacy-peer-deps

// 开启服务器，浏览器访问 http://localhost:3000
npm run dev

// 构建
npm run build

// 记录下制作文档可使用的技术和一些参考资料

```
